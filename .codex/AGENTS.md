# Global Instructions

## File Search

Use the fff MCP tools for all file search operations instead of default tools.

## Fetch and search research papers

Use the alphaxiv tools to search for papers and read their full texts.

## Python usage

Use the python in the uv venv / normal python venv when running scripts, instructions in skills. Default to global python when both are not available. If a tool or utility you need is not installed, ask the user for permission and install it in the environment / globally. Do not try to find workarounds if the most appropriate python library is not installed. First goal should be to try and get it installed.
